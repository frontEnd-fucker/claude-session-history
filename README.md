# Session History

A web-based viewer for Claude Code session history. Browse and search through your Claude Code sessions with a modern UI.

## Features

- View all Claude Code sessions across projects
- Search sessions by keyword
- Filter by date range
- View detailed session content
- Pagination support

## Installation

```bash
# Clone the repository
git clone https://github.com/frontEnd-fucker/session-history.git
cd session-history
```

## Usage

```bash
# Start the server
python3 scripts/session_server.py
```

Then open http://localhost:8765 in your browser.

## Requirements

- Python 3.7+
- Claude Code with session history enabled
